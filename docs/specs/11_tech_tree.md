# Spec 11 — Tech Tree & Upgrades

**Context:** The tech tree is the upgrade progression layer that unlocks new capabilities, improves existing systems, and eventually enables the prestige sequence. Three branches (Extraction, Processing & Crafting, Expansion) require Research Points generated by Research Labs. Before the Research Lab is built, three direct-purchase early upgrades are available. The tree is designed so that players can prioritize differently each run — Harvester Legacy bonus starters will follow a different path than Fleet Commander starters.

## 1. Early Direct-Purchase Upgrades (No RP Required)

Available from Phase 0. These are bought directly from the Trade Terminal, before any Research Lab is built.

| Upgrade | Cost | Effect |
|---|---|---|
| Drill Bit Mk.II | 50 CR | Player mine time: 1.5s → 0.75s |
| Cargo Pockets × 3 | 75 / 75 / 75 CR | +5 carry capacity per level (3 levels available) |
| Thruster Boots | 60 CR | +20% player move speed |

A crafted Drill Bit Mk.III (via Crafting Station schematic) is better than Mk.II, but Mk.II is the immediate early-game purchase.

| No Research Lab | 0 |

Note: RP generation is reduced by the productivity multiplier (see `06_consumption.md`). A 50% crew supply crisis halves RP output.

### Repair Drone

```
2.S  Repair Drone Unlock    500 RP + 0 CR      → Repair Drone available (800 CR purchase cost)   [req: 1.Q Refinery Drone Unlock]
```

**2.S Repair Drone:** Unlocks the Repair Drone for purchase (800 CR each). Repair Drones have Speed 90 px/s, carry capacity 0 (they carry repair materials internally), and accept REPAIR tasks only. They repair both harvesters AND cargo ships back to 100% efficiency. Used manually in Phase 3–4 and automatically in Phase 5 via the "Emergency Repair Protocol" tech. See `04_drone_swarm.md` drone roster and `07_logistics.md` §6.

**2.Z Advanced Fabrication:** Unlocks the Assembly Complex (3-slot apex production building). Requires the Fabricator unlock (2.X) as prerequisite. ### Warp Gate

The Warp Gate is unlocked by tech node 3.P (Warp Theory, 2,000 RP, requires 10 Void Cores produced) and must be **physically built** at a planet location. Key facts:

- **Build cost:** 20,000 CR + 50 Void Cores + 100 Alloy Rods (cost from `12_economy.md`)
- **Placement:** A3 (Void Nexus) is the primary Warp Gate location, but any planet can host one. Minimum two Warp Gates required (one at each end of a connection) to enable travel.
- **Function:** Enables **instant player travel** between any two planets that each have a Warp Gate. Unlike Jump Relays (which move *goods* between planets), Warp Gates move the **PLAYER** instantly — no travel animation, no fuel consumed for the player character.
- **Sector Complete trigger:** When all three planets (A1, Planet B, Planet C) are at Phase 4+ automation AND the Warp Gate is built at A3 and the Galactic Hub is complete, the Sector Complete sequence unlocks.
- **Does NOT replace Cargo Ships or Jump Relays:** Warp Gates only transport the player; all cargo still moves via Cargo Ships, Drone Freight Lanes, and Jump Relays.

See `09_planets.md` A3 section for placement context.
